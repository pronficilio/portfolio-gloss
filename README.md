# Technical Portfolio - Evidence of Work at Gloss Corporation

This repository gathers public-facing evidence of the technical work I performed across multiple projects during my time at Gloss Corporation.

The rights to the original software belong to the company. Because of that, this repository does not distribute source code, databases, credentials, internal assets, or proprietary implementations. Its purpose is to document real technical experience without exposing third-party intellectual property.

## Purpose

This repository serves as professional evidence of experience in:

- analyzing existing systems
- maintaining and evolving legacy applications
- working on PHP monoliths and Laravel applications
- integrating payments, authentication, messaging, and operational workflows
- producing technical documentation for portfolio, resume, and interview use

## What It Contains

Each folder represents a system or product line and includes:

- `README.md`: a high-level technical summary
- `portfolio.evidence.json`: structured evidence, stack details, modules, quality signals, and resume-ready bullet points

The content is written as technical documentation. It is not intended to reproduce the original implementation.

## What It Does Not Contain

- company source code
- secrets, keys, or operational configuration
- client data or sensitive information
- a runnable or redistributable version of the original systems
